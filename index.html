Utah Environmental Fluid Dynamics

LEMSv2 - Local Energy-Budget Measurement Stations

LEMS are solar powered, low-cost Energy-budget Measurement Stations, which have been designed and assembled at the University of Utah EFD lab.

Visit the LEMSv2 GitHub repository.

PREEVENTS

A fast-response wildland fire modeling framework for prediction and risk assessment.


Visit the PREEVENTS website.

QESWinds

QES-Winds is a fast response 3D diagnostic urban wind model written in C++ and uses NVIDIA's CUDA framework to accelerate a mass-conserving wind-field solver.


Visit the QESWinds Github repository.

UtahLSM - University of Utah Land Surface Model

UtahLSM is a lightweight, fast-response land-surface model suitable for use in large-eddy simulation (LES) codes.

Visit the UtahLSM Github repository.
